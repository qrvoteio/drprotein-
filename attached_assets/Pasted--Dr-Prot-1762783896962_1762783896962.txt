Общая информация	Название проекта	Dr.Protein — сайт функционального пептидного питания	 
Цель сайта	Образование, доверие, лидогенерация (на 1-м этапе); онлайн-продажи (на 2-м этапе)	 
Целевая аудитория	Дети, ЗОЖ-ники, офисные работники, домохозяйки, люди 30+, пожилые, биохакеры	 
Референсы	Soylent.com (структура, функционал), MyMatti.ru (натуральность, визуал)	 
Структура сайта	Главная страница	Лендинг с УТП, CTA, блоками: Наука, Для кого, Продукты, Блог	 
О пептидах / Наука	Объяснение пептидов, биодоступности, аминокислотного профиля, сравнение с обычным бульоном	 
Каталог продуктов	Бульоны, молочные коктейли, йогурты, перекусы (в будущем)	 
Для кого это?	Подстраницы: Для детей, Для спортсменов, Для офиса, Для семьи, Для 30+, Для 50+	 
Блог / Исследования	Статьи: научные, рецепты, интервью, истории	 
О нас	История бренда, производство, ценности, команда	 
Контакты	Форма обратной связи, телефон, email, соцсети, QR-код	 
Дизайн и UX	Стиль	«Технологичная натуральность» — минимализм + натуральные текстуры	 
Цветовая палитра	Основа: белый, светло-серый; акцент: синий (#2563EB) или зеленый (#10B981); дополнение: бежевый	 
Шрифты	Inter (Bold для заголовков, Regular для текста)	 
Фото	Лайфстайл-фото: продукт в использовании (офис, тренировка, семья)	 
Иллюстрации	Инфографика: биодоступность, синтез белка, возрастные изменения	 
Функционал	Главная страница	Hero-баннер, блок «Наука», «Для кого», «Продукты», форма обратной связи	 
Страница продукта	Галерея, состав, БЖУ, научное обоснование, сравнение с аналогами, отзывы	 
Форма обратной связи	Поля: Имя, Телефон, Email; интеграция с Bitrix24	 
Блог	Карточки статей с фильтрацией по рубрикам	 
Поиск по сайту	В шапке сайта	 
Подписка на рассылку	В подвале и после просмотра статьи	 
Технические требования	CMS	WordPress + Woocommerce (для будущих продаж)	 
Адаптивность	Mobile-first, корректная работа на всех устройствах	 
Скорость загрузки	>85 баллов в Google PageSpeed Insights	 
Интеграции	Google Analytics 4, Яндекс.Метрика, Bitrix24, SendPulse, Telegram/WhatsApp чат	 
Безопасность	SSL-сертификат, соответствие ФЗ-152	 
Контент-стратегия	SEO	Семантическое ядро: «пептидный бульон», «функциональное питание», «белковый дефицит»	 
Блог	2–3 статьи в неделю: наука, рецепты, кейсы	 
Соцсети	VK, Telegram, Дзен, MAX — образовательный и визуальный контент	 
Маркетинговые элементы	CTA	«Узнать больше», «Получить консультацию», «Скачать гид»	 
Лид-магниты	Гид по пептидному питанию, чек-лист «Белковый баланс», вебинар	 
Отзывы	Форма сбора отзывов после консультации/заказа	 
Юридические требования	Документы	Политика конфиденциальности, Пользовательское соглашение, Оферта (на 2-м этапе)	 
Информация о продукте	Состав, срок годности, условия хранения — на каждой странице продукта	 
Этапы и сроки	Этап 1	Прототип и дизайн

Этап 3	Наполнение контентом и тестирование
