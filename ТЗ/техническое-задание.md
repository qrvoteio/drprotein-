# Техническое задание: Dr.Protein

## Общая информация

**Название проекта:** Dr.Protein — сайт функционального пептидного питания

**Цель сайта:**
- **Этап 1:** Образование, доверие, лидогенерация
- **Этап 2:** Онлайн-продажи

**Целевая аудитория:**
- Дети
- ЗОЖ-ники
- Офисные работники
- Домохозяйки
- Люди 30+
- Пожилые
- Биохакеры

**Референсы:**
- Soylent.com (структура, функционал)
- MyMatti.ru (натуральность, визуал)

---

## Структура сайта

### 1. Главная страница
Лендинг с УТП, CTA, блоками:
- Наука
- Для кого
- Продукты
- Блог

**Функционал:**
- Hero-баннер
- Блок «Наука»

- Блок «Продукты»
- Форма обратной связи

Объяснение пептидов, биодоступности, аминокислотного профиля, сравнение с обычным бульоном

### 3. Каталог продуктов
- Бульоны
- Молочные коктейли
- Йогурты
- Перекусы (в будущем)

**Страница продукта:**
- Галерея
- Состав
- БЖУ
- Научное обоснование
- Сравнение с аналогами
- Отзывы

### 4. Для кого это?
Подстраницы:
- Для детей
- Для спортсменов
- Для офиса
- Для семьи
- Для 30+
- Для 50+

### 5. Блог / Исследования
Статьи:
- Научные
- Рецепты
- Интервью
- Истории

**Функционал:**
- Карточки статей с фильтрацией по рубрикам

### 6. О нас
- История бренда

### 7. Контакты
- Форма обратной связи

## Дизайн и UX

### Стиль
**«Технологичная натуральность»** — минимализм + натуральные текстуры

### Цветовая палитра
- **Основа:** белый, светло-серый
- **Акцент:** синий (#2563EB) или зеленый (#10B981)
- **Дополнение:** бежевый

### Шрифты
**Inter:**
- Bold для заголовков
- Regular для текста

### Фото
Лайфстайл-фото: продукт в использовании
- Офис
- Тренировка
- Семья

### Иллюстрации
Инфографика:
- Биодоступность
- Синтез белка
- Возрастные изменения
